Software Download

You can download the sources of my port via ftp://ftp.gmd.de/gmd/I5.RS/SRPC111.ZIP (about 540K) or via http://set.gmd.de/~mfg/SRPC111.ZIP. Use the -d option for unzipping. The binaries of ONC RPC for Windows NT for Intel processors are included in this distribution. ONC RPC for Windows NT currently comes with sources and makefiles for MS VC++ (all 32-bit versions). In addition also the binaries of version 1.09 compiled with Visual C++ for the Power PC (Thanks to Ray Drueke from Windward Technologies Inc.) and ALPHA (including the XDR fix, thanks to Wayne Seward) are available.
