ONC RPC for Windows NT

ONC RPC for Windows NT is a port of ONC RPC 4.0 (formerly called SUN-RPC) to Windows NT. It also runs on Windows95, while it was only tested with Windows NT 3.51 and NT 4.0.

General information about ONC RPC

General information about ONC RPC and the related XDR data encoding standard can be found in the RFCs 1831 (ONC RPC) and 1832 (XDR). You might also have a look in the book "Power Programming with RPC". There is also a small tutorial on SUNRPC out there. For a comparison of different RPC implementations (including ONC RPC) see John Barkley: Comparing Remote Procedure Calls. The sources of the original ONC RPC 4.0 can be downloaded e.g. from ftp://decuac.dec.com/pub/sources/rpc4.0-ultrix.tar.Z.
