Für die HMI Funktionalität ist SVG Unterstützung des Browsers nötig.

  • Firefox hat ab Version 2.0 SVG Support integiert.
  • Internet Explorer benötigt ein SVG Plugin von Adobe. Bisher ist mit dieser Kombination nur ein eingeschränktes Monitoring möglich.
  • Opera unterstützt ab Version 9.0 SVG und Javascript
